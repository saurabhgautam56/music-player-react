## ✨ Features

### 🎧 Player Controls

* ▶️ Play / Pause
* ⏭ Next / Previous
* 🔀 Shuffle mode
* 🔁 Repeat (Off / One / All)
* 🎚 Volume control
* ⏱ Progress bar with seek

---

### 📃 Playlist

* Scrollable playlist
* Active song highlight
* Auto-scroll to current song (desktop)
* Smart scroll (mobile friendly)

---

### 🔍 Search

* Real-time search filter
* Highlight matching text
* Works for both title & artist

---

### ⌨️ Keyboard Shortcuts

| Key   | Action         |
| ----- | -------------- |
| Space | Play / Pause   |
| →     | Next song      |
| ←     | Previous song  |
| ↑     | Volume Up      |
| ↓     | Volume Down    |
| M     | Mute / Unmute  |
| S     | Shuffle toggle |
| R     | Repeat mode    |

---

### 📱 Responsive Design

* Works on desktop & mobile
* Mobile tap-to-expand player
* Smooth UI transitions

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 CSS (Glass UI + Gradients)
* 🎵 HTML5 Audio API

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/saurabhgautam56/music-player-react.git
cd music-player-react
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the app

```bash
npm start
```

App will run on:

```
http://localhost:3000
```

---

## 🏗 Build for Production

```bash
npm run build
```

---

## 🌐 Deployment (Netlify)

1. Push project to GitHub
2. Go to Netlify
3. Import repository
4. Build command:

```
npm run build
```

5. Publish directory:

```
build
```
